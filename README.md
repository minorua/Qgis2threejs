Qgis2threejs plugin
===================

  This is a [QGIS](https://qgis.org/) plugin which visualizes DEM and vector data using MapLibre GL on web browsers.
The preview feature provides an interactive 2D/3D map view for spatial data.


Document
--------

  Online document: https://minorua.github.io/Qgis2threejs/docs/


Browser Support
---------------

  See [plugin wiki page](https://github.com/minorua/Qgis2threejs/wiki/Browser-Support).


Dependencies
------------

This plugin's preview feature is powered by the following JavaScript libraries:

* [MapLibre GL](https://maplibre.org/) - Interactive maps on the web
